### 실행방법
game 디렉토리내에서 Square Planet Escape.exe를 실행한다  
마우스움직임 시야  
방향키/스페이스로  플레이어 움직임  
좌클릭으로 공격,숫자키로 무기교환  
i키로 인벤토리 확인,p키로 게임실행 정지  
  
### 개발환경 설정방법
